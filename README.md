# GaodeMap
实现高德地图部分功能
=====
简要介绍
-----
1实现网络下的定位功能，获取经纬度，详细地址等<br>
2实现poi搜索关键字提示（listview）<br>
3实现点击地图某点时的定位显示<br>
4实现四种地图模式的切换（卫星，白昼，夜晚，导航）<br>
5实现四种出行方式的路线规划及在地图上的描绘（公交，驾车，骑行，步行）<br>
<br>
